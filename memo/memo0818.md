Router = Routing : 사용자가 접속한 URL에 해당하는 컨텐츠를 응답해주는 작업

URL vs URI
URL : 특정 위치의 Resource(자원)을 가리키는 의미이며, 주소
URI : 특정 위치를 나타내는 주소
URI : http://www.abc.com/index 파일명 포함 안 됨.
URL, URI : http://www.abc.com/index.html 주소, 파일명 언급
